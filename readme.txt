== Description ==

This is a plugin for random description,that is select a sentence with random in you like.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `rndesc.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place sentences for you like in `rndesc.php.lst`
3. Go to the Plugins -> Random Blog Description, Place sentences
